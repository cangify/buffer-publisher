关于软件

了解 Buffer发布器的用途、工作方式和官方网站。

Buffer发布器用于把图片、视频和文章内容批量发布到 Buffer。它会先将本地媒体转换为可被 Buffer 读取的公网直链，再提交到 Buffer 队列或按指定时间定时发布，适合集中管理多平台社交媒体内容。

  • 支持选择多个 Buffer 账号/频道后一次发布。
  • 支持临时媒体上传源、上传超时和本机 API 用量查看。
  • 支持图片/视频水印，以及本机 Ollama 自动生成标题建议。
官方网站 cangify.com
